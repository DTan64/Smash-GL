# Controls
#### Esc: Exit
#### f: Toggle first person mode
#### m: Toggle mouse mode
#### Perspective
  * Left arrow: rotate left
  * Right arrow: rotate right
  * Up arrow: rotate up
  * Down arrow: rotate down
  * 0: reset view angle

#### First Person Mode
  * w: move forward
  * s: move back
  * d: look right
  * a: look left
  * i: move up
  * k: move down

#### Mouse Mode
  * w: move forward
  * s: move back
  * Click and drag to look left/right

# Compile and Run
```make compile; make run;```
